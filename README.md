# Automated Shape Generator

## 📖 Overview

The **Automated Shape Generator** is a C++ console-based application designed to automatically generate a wide variety of geometric, numeric, and decorative patterns based on user-defined inputs. The application provides an interactive, menu-driven interface where users can choose from multiple pattern options, specify the desired size, and select a custom symbol (where applicable) to instantly generate the selected pattern.

The project includes **18 unique patterns**, including triangles, pyramids, diamonds, butterflies, Pascal's Triangle, spiral patterns, zig-zag patterns, hourglass, sandglass, cross, and several other geometric designs. The program is structured using multiple header files and modular functions, making the code organized, reusable, and easy to maintain.

Developed as a **Programming Fundamentals semester project**, this application demonstrates the practical implementation of core C++ programming concepts such as functions, loops, nested loops, conditional statements, arrays, modular programming, header files, and menu-driven program design. It serves as an excellent example of applying fundamental programming techniques to build an interactive and user-friendly console application.

## ✨ Features

- 🎯 Interactive menu-driven interface
- 🔢 Generates 18 different patterns
- 📏 Customizable pattern size
- ⭐ User-defined symbols
- 🧩 Modular code using multiple header files
- ⚡ Fast and efficient console output
- 💻 Beginner-friendly implementation
